2by2 Matrix Calculator

Matrix Calculator (JavaScript)

This plugin can calculate determinant, transpose, adjoint and inverse of any 2 cross 2 and 3 cross 3 matrix. This can be used in any educational blog or website.
You can contribute to this project on GITHUB if you found any bug or if you think that some feature should be added.
